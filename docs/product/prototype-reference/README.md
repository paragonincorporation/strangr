# Prototype visual references

These 1280×800 PNGs freeze the anonymous prototype's principal UI states before the React migration. They were captured from the local application on July 11, 2026.

They are visual references only. Anonymous, 18+, “no signup,” “no trail,” and automatic-report-skip language conflicts with the account-based beta plan and must not be ported as product policy.

Reproduce the baseline with:

```bash
npm run legacy:dev
```

Then open the printed legacy HTTP address. The active source implementation is under `public/`; the state descriptions and tested protocol are in `docs/architecture/prototype-baseline.md`.
